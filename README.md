# 🛒 Sistema de Cadastro de Produtos (Python)

Este projeto é um sistema simples de **CRUD de produtos** desenvolvido em Python, com o objetivo de demonstrar o uso combinado dos paradigmas:

- 🔵 Imperativo
- 🟢 Funcional

---

## 📌 Funcionalidades

O sistema permite:

- ✅ Cadastrar produtos  
- 🔍 Pesquisar produto por ID  
- ✏️ Atualizar produto  
- ❌ Remover produto  
- 📋 Listar todos os produtos  
- 💰 Calcular valor total do estoque  

---

## 🧠 Paradigmas Utilizados

### 🔵 Paradigma Imperativo

- Uso de variáveis mutáveis (`lista de produtos`)
- Estruturas de controle (`if`, `for`, `while`)
- Entrada e saída de dados (`input`, `print`)
- Manipulação direta de estado

### 🟢 Paradigma Funcional

- Funções puras (sem efeitos colaterais)
- Imutabilidade (funções retornam novas listas)
- Funções de ordem superior:
  - `map`
  - `filter`
  - `reduce`
- Separação entre dados e operações

---

## ⚙️ Tecnologias

- Python 3.x
- Biblioteca padrão (`functools`)
